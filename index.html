AWS Static Website

Amazon S3 • Amazon CloudFront • Amazon Route 53 • AWS Certificate Manager

Deployed on AWS

This static site is designed for deployment on Amazon S3 and global delivery via Amazon CloudFront, with a custom domain managed in Amazon Route 53 and HTTPS provided by AWS Certificate Manager (ACM).

View Project PDF

Fast

Content is cached at CloudFront edge locations for low latency worldwide.

Secure

HTTPS enabled using ACM-issued certificates attached to the CloudFront distribution.

Simple

Static assets (HTML, CSS, JS, images) are served from an S3 bucket configured for website hosting.

Test the Client-Side Script

Open the console or click the button to see a message from script.js.
